# E-Commerce для курса KTS "Начинающий разработчик" (Next.js)

## Реализовано

- Страница продукта /products/[id] (к примеру используйте /products/ynvrpepgr8vlyb4op8370h82)
- Просмотр нескольких изображений на странице товара
- Функционирующая корзина и страница корзины
- Анимации вау ого
- Просмотр нескольких изображений товара на его странице
- Фильтрация по цене
- Фильтрация по рейтингу
- Страница поиска товаров
- Список избранного
- Пагинация
