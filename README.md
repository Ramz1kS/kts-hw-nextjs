# E-Commerce для курса KTS "Начинающий разработчик" (Next.js)

## Скрипты

```
yarn run dev # для запуска
```
Репозиторий ДЗ4:
https://github.com/Ramz1kS/kts-hw

Деплой: [тут!](https://kts-ecommerce-nextjs.vercel.app)

## Реализовано

- Страница продукта /products/[id] (к примеру используйте /products/ynvrpepgr8vlyb4op8370h82)
- Просмотр нескольких изображений на странице товара
- Функционирующая корзина и страница корзины
- Анимации вау ого
- Просмотр нескольких изображений товара на его странице
- Фильтрация по цене
- Фильтрация по рейтингу
- Страница поиска товаров
- Список избранного
- Пагинация
- Эмуляция оплаты с промокодиками
- Похожие товары
- Состояние того, что товар добавлен в корзину
- Страница с категориями
Список промокодов:
```
    BULLY: 10%
    DONDAVIBES: 15%
    THISONEHEREISFAFA: 20%
    GOLDDIGGER: 20%
    24HOURS: 24%
    ALLTHELOVE: 30%
    BULLETPROOF: 40%
    PAYPERVIEW: 50%
```
